Date: June 26, 2017.
This is the full release of the dataset for the AMIA shared task - subtask 3 - concept normalization.

The folder contains four concept normalization training files for the AMIA shared task.

Format of the data:

ID <-tab-> Text <-tab-> MEDDRA PT

Total of 4 files are present in the folder with 6,649 annotated instances.
